Project Synopsis

Ingenium is an Android application which gamifies the language learning process. The application allows tutors to create custom datasets of language courses they are teaching which the student can import into the application. The application generates games for the student to play through, augmenting language learning materials and adding to the learning experience.

While there are many language learning applications in the marketplace, there is a niche for an application which allows customisable content where the tutor controls the syllabus.
