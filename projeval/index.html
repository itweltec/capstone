Project System Evaluation


Our Client, Robert Sutcliffe, had a problem.

Suggesting a project management system to Capstone Project students would help them get started on their projects and encourage consistency. This would save everyone, staff and students, time and effort. But what project management system to suggest? And how to help them use it? As the WelTec Capstone Project course coordinator, these were important questions...

That's where we came in

The Work

After consulting with our Client, we realised that the project would need to be divided into two parts:

Part 1 - Finding project management systems which were suitable for Capstone Project needs, and creating a Recommendations Report of the shortlisted systems for our Client.

Part 2 - Developing instructions on how to use the project management system selected by our Client.

After thoroughly evaluating and comparing 50 different project management systems, we wrote up a Recommendations Report which recommended the top four options. The Client used this information to choose one project management system. We then developed Procedural Instructions on how to use it.

The Outcome

The comprehensive evaluation and comparison process, and the information in the Recommendations Report, allowed the Client to choose Basecamp as the project management system which is the most suitable for the needs of Capstone Project staff and students.

Five Procedural Documents were developed - one for each identified user type - and delivered to the Client as PDFs. An editable version of the instructions, and details on how to update them, were also presented to the Client.

The work that we have done during this project means that our Client, the WelTec Capstone Project course coordinator, can confidently suggest Basecamp to students. This is because he knows that it is suitable for their needs, and that the instructions will help the students get familiar with Basecamp quickly. If more students are using Basecamp, then it will be easier for staff to assess their work and provide advice because they will not need to learn a new project management system for every project team.
