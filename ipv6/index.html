Auditing

Our team will have an audit plan which will define scope and scheduling of the auditing which need to be validated. This plan will identify device to be audited. The audit will be done to showcase the validation of the IPv6 network, whether the devices or IOS are IPv6 compliant and an audit will be done to show which IP range is being used or should have been used.

Validation/Testing

The testing will be done to show case the validation of the IPv6 network, whether the devices are UpToDate or not and an audit will be done to show which IP range is been used or could be used. Test may run multiple time. This test will also produce multiple cases. Test will be recorded with the help of screen shots and the detail will be presented in the table format. The test plan will show current results as per live scenario.
