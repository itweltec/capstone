Recommendations Report

  1. Home
  2. Recommendations Report

After selecting the shortlist, we needed to create a Recommendations Report. The Recommendations Report was used by our Client to choose the project management system which is most appropriate for the needs of Capstone Project staff and students. This report contained details of our evaluation and comparison methodology, and the results.

The report included information about each of the four shortlisted project management systems, including the advantages and disadvantages of each system and how they compared to one another for each of the evaluation categories. It include recommendations from the project team about which project management system was the most suitable. This was a challenge because each of the shortlisted systems had pros and cons, and which one was best for Capstone Project users depended on what was prioritised e.g. cost vs user friendliness vs project management tools.


The Chosen One:

After reading our report, the Client chose Basecamp. This was because Basecamp is the most user friendly, has the best range of collaboration tools, and provides an education discount which means that the full version is free for students and teachers.

The Shortlist

From 50 to 4. These are the four project management systems that made it to the Recommendations Report

Avaza

Avaza was the best option for project management tools and access control, and has a comprehensive free plan. However the large number of options can be intimidating and makes Avaza time consuming to learn how to use. It's collaboration tools are also very limited.

Basecamp

Basecamp focuses on a simple user interface and collaboration tools. It has basic project management, access control, and auditing functionality. The full version can be used for free by teachers and students. The full version lets the course coordinator manage student projects from a central location.

Freedcamp

Freedcamp is a good 'all-rounder'. It has a comprehensive free plan that lets the course coordinator manage student projects from a central account, and solid project management and access control functions. However, the free plan only records 7 days worth of project activity and the user interface can be confusing.

ZenHub

ZenHub is fully integrated with GitHub and is an ideal choice for software development projects or project which use the Agile methodology. However, a GitHub account is needed to use ZenHub, it has limited communication tools, and is difficult to use with non-Agile approaches to project management.
